# Swagger

Este directorio contiene la configuración y archivos relacionados con Swagger/OpenAPI para la documentación interactiva de la API.

## Propósito
- Configuración de Swagger UI
- Especificaciones OpenAPI
- Documentación interactiva de endpoints
- Schemas y modelos de datos

## Archivos típicos
- `swagger.config.ts` - Configuración de Swagger
- `swagger.json` - Especificación JSON de la API
- `swagger.yaml` - Especificación YAML de la API
- `schemas/` - Esquemas de datos reutilizables
- `examples/` - Ejemplos de requests/responses

## Uso
Los archivos en este directorio se utilizan para:
- Generar documentación automática de la API
- Proporcionar interfaz web para probar endpoints
- Validar requests y responses
- Generar código cliente automáticamente
